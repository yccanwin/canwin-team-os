## 四分类定义

| 分类 | 允许动作 |
| --- | --- |
| 保留 | 结构和业务语义符合 4.0，只补权限、索引和接口 |
| 扩展 | 保留旧字段，以新增兼容字段、事件表或适配视图承载 4.0 |
| 只读 | 停止旧业务新写入；有业务价值的历史数据由 4.0 新入口查询，不要求旧入口继续在线 |
| 淘汰候选 | 关闭旧入口和新写入，完成依赖、备份和可恢复验证；物理删除必须另行授权 |
