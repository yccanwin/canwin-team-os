## 2026-06-26 — 微交互动效

**文件**: `src/hooks/useCountUp.ts`(新), `src/index.css`, `src/pages/Dashboard/KPISection.tsx`

### 一、useCountUp hook (新)
- `requestAnimationFrame` + ease-out-quart (`1-(1-t)⁴`) 缓动
- 参数: `end: number`, `duration?: number` (默认 800ms)
- 初始计数从 0→end；end 变化时从旧值→新值平滑过渡
- 用 `useRef` 跟踪起始值，避免重复触发

### 二、全局动效 CSS
- **card-hover** — `translateY(-2px)` + `box-shadow` 加深，`cubic-bezier(0.34,1.56,0.64,1)` 弹性曲线
- **btn-press** — `:active` 时 `scale(0.97)`，150ms
- **fadeInUp** — 列表项入场 `opacity:0→1 + translateY(12px→0)`，400ms ease-out
- **stagger-1..10** — 逐项动画延迟 (50ms~500ms)，通过直接类名控制
- **slideUp** — 通知弹入 `translateY(100%→0)`，300ms ease-out

### 三、KPISection 数字滚动
- `thisMonthIncome` / `inProgressCount` / `goalRate` / `totalAssets` 接入 useCountUp
- 动画值通过 `toLocaleString()` 格式化，团队等级保持静态

**构建**: vite build 零错误（502ms）
