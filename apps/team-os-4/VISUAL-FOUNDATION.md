# Team OS 4.0 视觉基础层

本层为全新设计，不复制或引用 3.0 源码、CSS、路由、RPC、状态仓库或权限逻辑。

| 4.0 基础件 | 通用职责 | 禁止承载 |
| --- | --- | --- |
| `StatusBadge` | 状态语义与紧凑标记 | 业务状态字典 |
| `ProgressBar` | 0–100 限幅和无障碍进度语义 | 积分、目标计算 |
| `KPICard` | 指标信息层级 | 数据查询和跳转权限 |
| `EmptyState` | 空内容说明与可选操作 | 业务默认动作 |
| `Modal` | 遮罩、ESC、滚动锁 | 表单、鉴权逻辑 |
| `ConfirmDialog` | 确认、取消和等待态 | 删除、审批实现 |
| 4.0 壳层 CSS | 深绿侧栏、金色品牌、浅色内容区和移动适配 | 导航授权、旧 Layout 行为 |

P1 页面可以组合这些基础件；岗位可见性、导航和动作授权必须来自 4.0 新权限合同。
